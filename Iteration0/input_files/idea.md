**Title**: Pareto-Optimal Calibration of GPU-Accelerated PM Dynamics: Balancing Temporal Integration and Mesh Resolution for Cosmological Emulators

**Description**: This research quantifies the computational trade-offs of an NVIDIA Warp-based Particle-Mesh simulation by constructing a Pareto-optimal front of wall-clock time versus $P(k)$ accuracy across varying time-step counts (50–100) and mesh resolutions ($512^3$ vs $1024^3$). To decouple ZA-induced power deficits from numerical artifacts, we compute a baseline transfer function between our ZA-generated ICs and the Quijote 2LPT initial conditions at $z=127$, allowing for the isolation of post-initialization numerical convergence. By mapping the $P_{warp}(k)/P_{Quijote}(k)$ ratio within the $0.01–1\ h/\text{Mpc}$ range, we identify the specific temporal and spatial resolution settings that minimize runtime while maintaining the target $<5\%$ error threshold for $k < 0.3\ h/\text{Mpc}$. This analysis provides a rigorous calibration framework for the Warp framework, establishing the minimum computational budget required to achieve high-fidelity power spectra. The final result is a performance-accuracy map that enables developers to optimize GPU resource allocation for large-scale cosmological parameter inference.
